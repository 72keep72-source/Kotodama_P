コトプロ｜ルールブック作成キット README
------------------------------------

このフォルダには、
「言霊のプロトコル（コトプロ）」上で動作する
ルールブック（シナリオ）を作成するための
フォーマットと資料が含まれています。

本キットは実験用です。
内容は予告なく変更されることがあります。


■ 内容物
- シナリオ作成ガイド（PDF）
- シナリオフォーマット（txt）
- シナリオの書き方例（txt）
- このREADME


■ 使い方
- フォーマットをコピーして自由に編集してください
- 完成していなくても問題ありません
- 特別なツールは不要です（テキストエディタで読めます）


■ 提出方法について
完成したルールブックの受け渡しは、
『データ便サービスの利用を原則』とします。

・メガファイル便
・ギガファイル便
・その他、一般的なデータ転送サービス

いずれを使用しても構いません。

連絡手段としては、
現時点では X（旧Twitter）のDMを想定していますが、
プラットフォームの仕様変更等により、
将来変更される可能性があります。

最新の連絡方法については、
コトプロ公式ページまたは案内文をご確認ください。


■ 実装について
- すべての作品が実装されるわけではありません
- 内容確認および挙動確認の上、
  主宰の判断で実装されます
- 実装時に、表現・数値・構造などを
  調整する場合があります


■ 作品の扱いについて（重要）
このキットを使って作成したルールブックは、
【作者本人の作品です。】

主宰に送付した場合でも、
- 実装されなくても問題ありません
- 実装されなかった作品を、
  作者自身が公開・紹介・改変・発展させることは
  完全に自由です

コトプロに送ったからといって、
作品の権利が主宰に移ることはありません。

「送りつけて終わり」でも、
「試しに作っただけ」でも、
まったく問題ありません。


■ 注意
- 本キットの利用において、
  金銭のやり取りは一切発生しません
- 本キットは「作ってみたい」を試すためのものです
- 読むだけの利用も歓迎します


コトプロは、
ルールブックや世界観を共有して遊ぶ
テキストRPGの実験的プラットフォームです。

うまく作れなくても大丈夫です。
少し試しただけでも、十分な参加です。
